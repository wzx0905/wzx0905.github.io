如果想在浏览器中具体看到一个盒子占有的实际位置，需要设置盒子可以实体化的三属性。

| 属性名           | 属性值         | 说明               |
| ---------------- | -------------- | ------------------ |
| width            | px单位的数值   | 定义元素占有的宽度 |
| height           | px单位的数值   | 定义元素占有的高度 |
| background-color | 颜色名、颜色值 | 定义元素的背景颜色 |
