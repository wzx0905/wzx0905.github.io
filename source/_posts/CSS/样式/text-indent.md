作用：设置段落首行是否进行缩进。

属性值

| 属性值 | 说明                                                |
| ------ | --------------------------------------------------- |
| px单位 | 表示首行缩进多少像素                                |
| em单位 | 首行缩进几个中文字符的位置                          |
| 百分比 | 表示缩进文字所在标签的父级标签的width属性值的百分比 |

实际工作中，最常使用 em 为单位的属性值。

属性值区分正负，正数表示向右缩进，负数表示向左缩进。
