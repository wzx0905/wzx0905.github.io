作用：设置文本整体是否有线条的修饰效果。

属性值

| 属性值       | 说明                            |
| ------------ | ------------------------------- |
| none         | 没有修饰，大部分标签的默认值    |
| overline     | 上划线                          |
| line-through | 中划线，删除线，del标签的默认值 |
| underline    | 下划线，a标签的默认值           |
