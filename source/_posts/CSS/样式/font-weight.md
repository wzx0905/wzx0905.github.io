作用：设置文字是否加粗显示。

属性名：font-weight，属于 font 属性的一个单一属性。

属性值有两种方式：单词类型、数字类型。

单词类型

| 属性值  | 说明                                |
| ------- | ----------------------------------- |
| normal  | 默认值，定义标准的字体              |
| bold    | 定义粗体字符，b、strong标签的默认值 |
| bolder  | 定义更粗的字体                      |
| lighter | 定义更细的字体                      |

**数字类型**

100-900 之间的整百数字。

数字越大，文字显示越粗。

其中 400 等价于 normal，700 等价于 bold。
