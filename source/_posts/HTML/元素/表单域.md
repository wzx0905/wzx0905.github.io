HTML表单域使用form标签进行定义。

form标签是一个功能性标签，填写的表单信息要想正确的提交到后台服务器，必须放在一个form标签之内。

form标签为双标签，容器级标签。

### 标签属性

form标签通过对应属性规定提交数据的方法和提交位置。

| 属性名 | 属性值     | 描述                                        |
| ------ | ---------- | ------------------------------------------- |
| action | url        | 指定接收并处理表单数据的服务器程序的url地址 |
| method | get/post   | 用于设置表单数据的提交方式                  |
| name   | 自定义名称 | 规定表单的名称                              |
